Movie APi

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Response
Get a list of movies /movies A JSON object holding data about all the movies
Return director bio /directors/[director name] Return data about a director (bio, birth year, death year) by name
Return data about a single movie /movies/[title of movie] Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
Return data about a genre /movies/genres/[genre] Return data about a genre (description) by name/title
Allow new users to register /sign-up Allow new users to register for a new account
update user info /users/[username]/update Allow users to update their user info (username)
Add to favorites /users/[username]/favorites/add-favorites Allow users to add a movie to their list of favorites
Remove a movie from their list of favorites /users/[username]/favorites/delete-favorites Allow users to remove a movie from their list of favorites
Allow existing users to deregister /users/[username]/deregister Allow users to de-register
